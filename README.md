> ripple effect for vue.js app -- only depend on vue2

[Live Demo](https://wsqviva.github.io/vue-haru-ripple/)

## Install
install via yarn
```
$ yarn add vue-haru-ripple
```

## Usage
```html
<vue-haru-ripple class="ripple">
  <div>
    <span>my ripple</span>
  </div>
</vue-haru-ripple>
```
